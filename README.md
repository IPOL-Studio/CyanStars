## CyanStars 项目简介

CyanStars 中文译名暂定为“翠蓝星穹”，是一个非商业同人音乐游戏，计划将开源

本游戏以Vsinger相关的虚拟歌手为基础进行展开

项目尚处于早期阶段（其实根本没开动过）
